sait3_site — обновлённый персональный сайт

Файлы:
- index.html — главная страница (SPA-подобная навигация)
- styles.css — стили и адаптивность
- scripts.js — небольшая логика: навигация, аккордеон, превью и пазл

Как запустить:
1. Откройте `index.html` в браузере (двойной клик) или
2. Запустите локальный сервер в папке `sait3_site` (например, Python):

```bash
# в PowerShell
python -m http.server 8000
# затем откройте http://localhost:8000
```

Если нужно, могу собрать версию в один файл (встраивание CSS/JS) или добавить деплой на GitHub Pages.
